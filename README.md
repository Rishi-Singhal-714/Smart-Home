# 🏠 Smart Home Automation

A voice-controlled IoT-based Smart Home Automation System built using **ESP32**, enabling seamless control of lights and fans through both **Google Assistant voice commands** and a **web-based dashboard**.

## ✨ Features

- 🎙️ Voice control using Google Assistant
- 💡 Turn lights ON/OFF remotely
- 🌬️ Control fan ON/OFF and adjust fan speed
- 🌐 Real-time web dashboard for manual control
- ⚡ Fast and responsive ESP32 communication
- 🔄 Simultaneous support for voice and browser-based control

## 🚀 Project Overview

Developed a voice-controlled Smart Home Automation System using **ESP32**, allowing users to control lights and fan speed through both **Google Assistant** and a **browser-based web interface**.

The system integrates embedded firmware with a Flask-powered web server to provide reliable, low-latency communication between hardware and software. Users can switch appliances, adjust fan speed, and monitor device states in real time from any connected device.

The embedded layer was developed in **Embedded C**, while the web interface was built using **Python (Flask), HTML, CSS, and JavaScript**, enabling seamless interaction between the ESP32 and the control dashboard.
